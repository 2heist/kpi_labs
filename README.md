# Car Rental System

This project contains data models and UML diagrams for a car rental system.

## Project structure
- `docs/diagrams/` — PlantUML files (`.puml`) and images (`.png`)
- `src/` — source code (optional)
- `.gitignore` — ignored files list
- `README.md` — project info

## How to view diagrams
1. Open `.puml` files in VS Code with the PlantUML extension.
2. Press `Alt + D` to preview the diagram.
3. Or generate images with:
   ```bash
   plantuml docs/diagrams/er_model.puml
   plantuml docs/diagrams/uml_model.puml
